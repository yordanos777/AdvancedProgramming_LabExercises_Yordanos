NOTEPADFX APPLICATION

Description:
This is a simple text editor application created using JavaFX.
Users can create, open, edit, and save text files.

Concepts Used:
- OOP Fundamentals
- System Programming
- File Handling
- Multithreading
- JavaFX GUI

Files:
- NotepadFX.java

Features:
- Open text files
- Save text files
- Word and character counter
- Clear text option
- Menu system

How to Run:
1. Run NotepadFX.java
2. Use File menu to open or save files
3. Use Edit menu for word count

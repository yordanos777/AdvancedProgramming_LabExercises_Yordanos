# JavaFX Chat Application

## Overview

This project is a JavaFX Client-Server Chat Application built using Java Socket Programming, JavaFX GUI, and MySQL Database.

The application allows multiple clients to connect to a server and exchange messages and files in real time.

# Features

- Real-time client-server communication
- Multiple client support
- JavaFX graphical interface
- MySQL chat history storage
- Load last 10 messages on startup
- File transfer support
- Object serialization
- Multi-threading using Thread class
- Separate client and server applications

# Technologies Used

- Java
- JavaFX
- Socket Programming
- MySQL
- JDBC
- XAMPP / phpMyAdmin
